# Customer Churn Prediction System

## Overview
This project predicts customer churn using Machine Learning and provides an interactive dashboard built with Streamlit.

## Features
- Churn probability prediction
- Business risk categorization
- Interactive analytics dashboard
- ROC-AUC model evaluation

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
